h2. Description

playujs-examples is an playframework app using "playujs":https://github.com/rafaelfelini/playujs module

h2. Running the example

bc. git clone git@github.com:rafaelfelini/playujs-examples.git
cd playujs-examples
play deps
play run
Browse: http://localhost:9000/

Or browse to "http://playujs.herokuapp.com/":http://playujs.herokuapp.com/ to see the live sample.

Enjoy!
